**MSYS2** Git for Windows (gitbash) ======================================================= MSYS2 ======================================================= - http://www.msys2.org/ ## pacman - https://wiki.archlinux.org/index.php/pacman Option | 説明 -------------------- | ----------------- -S -h | help -Syy | forcely update database all -Syyu | forcely update database all & update packages -S package | install -R package | uninstall -Ss [pattern] | search packages -Sg [package group] | print package group -Si [package] | print information -Sl | print list -Qi installed_package | info of the package -Ql installed_package | list files in the package ## paccache 最新の３つのバージョンを残してパッケージのキャッシュを削除 ~~~ paccache -r ~~~ ## /etc/pacman.conf ~~~ #XferCommand = /usr/bin/curl -C - -f %u > %o XferCommand = /c/Windows/System32/curl -C - -f %u > %o [mingw64] Server = https://MIRROR_SERVER/mingw/x86_64/ ~~~ ## disable POSIX-to-Windows path conversion 以下のいずれかの方法がある 1. パスの先頭の'/'を'//'に変更して記述する ~~~ //c/dir/fname ~~~ 2. MSYS_NO_PATHCONVを設定する ~~~ export MSYS_NO_PATHCONV=1 ~~~ 3. MSYS2_ARG_CONV_EXCLを設定する ~~~ export MSYS2_ARG_CONV_EXCL="*" ~~~ ## adb | Command | Description | | --- | --- | | adb push files //sdcard/Download | push files | ## stdc++ man pages - https://github.com/jeaye/stdman ファイル名にstd::...を使用しているのでMSYS2のgitを使う ~~~ /usr/bin/git clone https://github.com/jeaye/stdman.git ~~~ Cygwin ======================================================= ## DLLの読み込みに失敗する場合 ~~~ C:\cygwin\bin\dash.exe -c '/usr/bin/rebaseall -v' ~~~ ## X serverのインストール - xorg-server - xinit ## X serverの起動 スタートメニュー -> XWin Server ## テキストファイル形式 UTF8, DOS形式 ## 現在のパスをWindows形式の絶対パスに変換 ~~~ cygpath -am `pwd` ~~~
